# WorkFlow App — GitHub Pages Deployment Guide
### Install on Android in ~10 minutes (Free, No coding needed)

---

## WHAT YOU NEED
- A **GitHub account** (free) → https://github.com/signup
- A **computer or phone** with a browser
- Your **Android phone** with Chrome

---

## STEP 1 — Create a GitHub Account (skip if you have one)

1. Go to **https://github.com/signup**
2. Enter your email, create a password, pick a username
3. Verify your email

---

## STEP 2 — Create a New Repository

1. Log in to GitHub
2. Click the **"+"** icon (top-right corner) → **"New repository"**
3. Fill in:
   - **Repository name:** `workflow-app`
   - **Description:** `WorkFlow Planner App`
   - **Public** (must be public for free GitHub Pages)
4. Click **"Create repository"**

---

## STEP 3 — Upload the App Files

1. On your new repository page, click **"uploading an existing file"** link
   (or go to: `https://github.com/YOUR_USERNAME/workflow-app/upload/main`)
2. **Drag and drop ALL 5 files** from the zip folder:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`
3. Scroll down, click **"Commit changes"**

---

## STEP 4 — Enable GitHub Pages

1. Go to your repository → click **"Settings"** tab (top menu)
2. In the left sidebar, click **"Pages"**
3. Under **"Source"**, select:
   - **Branch:** `main`
   - **Folder:** `/ (root)`
4. Click **"Save"**
5. Wait 1–2 minutes. GitHub will show a green banner:
   **"Your site is live at https://YOUR_USERNAME.github.io/workflow-app/"**

---

## STEP 5 — Install on Android

1. Open **Chrome** on your Android phone
2. Go to: **https://YOUR_USERNAME.github.io/workflow-app/**
3. The app loads — wait for it to fully appear
4. You'll see a banner: **"Add WorkFlow to Home screen"** → Tap **"Install"**

   **If no banner appears:**
   - Tap the **⋮ three-dot menu** (top-right in Chrome)
   - Tap **"Add to Home screen"** or **"Install app"**
   - Tap **"Add"**

5. The **WorkFlow** app icon now appears on your home screen!
6. Open it — it runs **full-screen like a native app**, no browser bar

---

## DONE! 🎉

Your app is now:
- ✅ On your home screen like a real Android app
- ✅ Works offline after first load
- ✅ Your data saves locally on your phone
- ✅ Free forever — no hosting costs on GitHub Pages

---

## UPDATING THE APP LATER

If you want me to add features and give you an updated version:
1. Go to your GitHub repository
2. Click on `index.html`
3. Click the **pencil icon** (edit) or delete & re-upload
4. Commit the changes
5. Wait 1–2 minutes — your phone app auto-updates!

---

## TROUBLESHOOTING

**"Page not found" error?**
→ Check Settings → Pages → make sure branch is set to `main`

**App doesn't install?**
→ Make sure you're using Chrome (not Samsung browser)
→ Try clearing Chrome cache and reloading

**Changes not showing?**
→ GitHub Pages can take 2-5 minutes to update
→ On your phone: open the app → pull down to refresh
